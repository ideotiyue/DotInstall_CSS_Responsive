レスポンシブウェブデザイン

mediaクエリを用いて、viewportの幅に応じてスタイルを切り替える。スタイルが切り替わるときのviewportの幅をブレイクポイントと呼ぶ。
複数パターン作る場合、CSSではあとから書いたものが優先される仕組みを使って、共通のスタイルは先に書いて必要な分だけあとから上書きしていく、というテクニックがよく使われる。

▼Viewportの幅の最低/最大値を指定するmediaクエリ

@media (min-width: 600px) and (max-width: 800px) {　}

※()の前後には、半角空白が必ず必要な点に注意

モバイルファースト

このファイルで使用しているテクニックは、 CSS はあとから書いたもので上書きされるという仕組みを使っている。
ブラウザ幅の条件を小さい順に書いたときだけ使えるという点に注意。
このようなメディアクエリはモバイルサイトのスタイルから順番に作っていくという意味でモバイルファーストと呼ばれる。
